## ⚡ About The App

**OhmLaws** is a mobile application designed to help electrical students master their trade. It provides a comprehensive collection of questions, answers, and suggestions available completely offline.

Whether you are preparing for exams or need a quick reference in the workshop, OhmLaws is ready without an internet connection.

## 🚀 Features

* **100% Offline:** No internet connection required. Works anywhere.
* **No Login Needed:** Just open the app and start studying immediately.
* **Question Bank:** Curated Q&A for ITI Electrician syllabus.
* **Lightweight:** Optimized for performance on all Android devices.

## 📱 Installation

Since this app is not yet on the Play Store, you can install it directly from GitHub:

1.  Click the  <a href="https://github.com/roniui/ohmlaw-app/releases/latest">
    <img src="https://img.shields.io/badge/Download_Latest_APK-3DDC84" alt="Download APK">
  </a>  button above.
2.  Download the `.apk` file (e.g., `OhmLaws-v1.0.x.apk`) from the "Assets" section.
3.  Open the file on your Android device.
4.  If prompted, allow **"Install from Unknown Sources"** (this is required for apps outside the Play Store).
5.  Install and enjoy!
